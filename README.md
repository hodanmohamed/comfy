# Comfy Boots

Comfy Boots presents a responsive multi-page footwear website, expertly built with HTML, CSS, and JavaScript. It allows users to explore the latest collection, find style inspiration, and enjoy a seamless shopping experience. Whether for comfort or fashion, Comfy Boots ensures a visually appealing and user-friendly platform for all footwear enthusiasts.

![ComfyBoots](https://github.com/hodanmohamed/comfy/assets/100366164/b5ab622a-5d26-4a3f-b00f-1588ed7ff96a)


## Tech Stack

**Languages:** HTML, CSS, JavaScript

## Run Locally

Clone the project

```bash
  git clone https://github.com/hodanmohamed/comfy.git
```

Go to the project directory

```bash
  cd comfy
```

Install dependencies

```bash
  npm install
```

Start the server
